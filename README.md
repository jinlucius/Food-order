## 🧠 Project Overview

This application allows users to:

* Browse food items by category
* Add/remove items from cart
* View real-time cart updates
* Place orders with delivery details

It uses **React Context API** for global state management and provides a **smooth, dynamic user experience**.

## 🏗️ Architecture

* Component-based structure
* Centralized state using Context API

### 🔁 Data Flow

ExploreMenu → Home (state) → FoodDisplay → FoodItem → Cart → PlaceOrder
